Скинуть все файлы в домены на openserver
Запустить open server
Открыть проект через Open server
Из папки DB загрузить базу данных в PHPMyAdmin
В папке Config изменить файл db.php настроить подключение к бд
На главной странице нажимаем кнопку "Проверить"
Пишем свой PHP-код и проверку
На странице "Управление" идёт работа с базой данных, которая хранится в папке DB





Пути к файлам:
Проверка PHP-кода - views\site\code.php
Создание строчки в бд - views\temi\create.php
Изменение строчки в бд - views\temi\update.php
Отоброжение информации из бд - views\temi\index.php
Форма для работы с бд - views\temi\_form
